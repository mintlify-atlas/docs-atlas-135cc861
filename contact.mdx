## Send a message

You can submit a message directly through the contact form on the [live portfolio site](https://afnan-altaf.github.io/portfolio/). Alternatively, send an email to [afnanaltafjamber@gmail.com](mailto:afnanaltafjamber@gmail.com) and you will receive a reply as soon as possible.

The contact form submits a `POST` request to the Formspree endpoint `https://formspree.io/f/mwverpag` with the following fields:

| Field     | Type   | Description              |
| --------- | ------ | ------------------------ |
| `name`    | string | Your full name           |
| `email`   | string | Your email address       |
| `message` | string | The message you are sending |
